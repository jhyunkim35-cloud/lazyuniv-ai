DATA MODEL (Firestore + IDB mirror):
- users/{uid}/gamification/state (single doc, document id is literally "state")
- Fields:
  - streak: number (consecutive days reviewed, default 0)
  - lastReviewDay: 'YYYY-MM-DD' | null (most recent day with at least 1 review)
  - xp: number (default 0)
  - todayDoneCount: number (cards reviewed today; reset when new day)
  - todayYmd: 'YYYY-MM-DD' (when todayDoneCount was last touched; helps reset on day rollover)

NEW FILE /public/js/gamification.js exposing window.* functions:
- getGamificationState() -- read current state, lazy-create with defaults if missing. Returns the state object. FS first, IDB fallback.
- saveGamificationState(state) -- write to FS + IDB with merge:true.
- onCardReviewed(quality) -- update state based on a review event:
  - today = todayYmd()
  - if state.todayYmd !== today: reset state.todayDoneCount=0, state.todayYmd=today (day rolled over)
  - if state.lastReviewDay is null: streak = 1
  - else if state.lastReviewDay === today: streak unchanged (already counted today)
  - else if state.lastReviewDay === yesterday: streak += 1
  - else: streak = 1 (broken)
  - state.lastReviewDay = today
  - state.xp += (quality >= 3 ? 10 : 5)
  - state.todayDoneCount += 1
  - saveGamificationState(state)
  - return state
