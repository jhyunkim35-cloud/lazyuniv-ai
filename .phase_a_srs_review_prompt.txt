═══ Part 1: New file public/js/srs_review.js ═══

Module that owns the review UI. Functions:

- enterReviewMode(folderId | null) — fetch due cards (getDueCards), build review queue, switch to review screen
- renderReviewCard(card) — show the front (section title + maybe note title), with "정답 보기" button
- showAnswer(card) — reveal the section's content (look up the note + the section by title)
- onGradeClick(quality) — call gradeCard + onCardReviewed; if more cards, advance; if done, show summary
- exitReviewMode() — return to home or note

Quality scale (Duolingo-friendly, mapped to SM-2 0-5):
- "완전 모름" → quality 0  (red, full reset)
- "어려웠음" → quality 2  (orange, smaller interval)
- "괜찮음"   → quality 4  (green, normal interval)
- "쉬움"     → quality 5  (blue, big interval boost)

Show after each grade:
- "다음 복습: N일 후" (use computed card.nextReviewYmd diff from today)
- XP gained badge animation

When all cards done, show summary screen:
- "오늘 N장 복습 완료!" + total XP earned + new streak count + "확인" button to go home
